# Getting Started

Welcome to Koban! This is your first space.

## Quick Start
- Open the Kanban board by clicking the space or pressing `CMD+Shift+K`
- Create tasks with `CMD+Shift+T`
- Select code, right-click → "Create Koban Task from Selection"
