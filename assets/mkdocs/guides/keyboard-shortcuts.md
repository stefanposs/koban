# Keyboard Shortcuts

Koban is designed for keyboard-first usage. All shortcuts work when the Kanban board is focused.

## Global Shortcuts

| Shortcut | Action |
|----------|--------|
| ++cmd+shift+k++ / ++ctrl+shift+k++ | Open Kanban Board |
| ++cmd+shift+t++ / ++ctrl+shift+t++ | New Task |

## Board Navigation

| Key | Action |
|-----|--------|
| `J` | Move selection down |
| `K` | Move selection up |
| `Space` / `Enter` | Open selected task file |

## Task Actions

| Key | Action |
|-----|--------|
| `C` | Create new task (quick-add) |
| `M` | Create new meeting |
| `A` | Archive selected task |
| `1` | Move task to column 1 (To Do) |
| `2` | Move task to column 2 (In Progress) |
| `3` | Move task to column 3 (Review) |
| `4` | Move task to column 4 (Done) |

## Modal Shortcuts

| Key | Action |
|-----|--------|
| `Escape` | Close all modals |

!!! tip "Mouse is Optional"
    You can manage your entire Kanban board without touching the mouse. Use `J/K` to navigate, number keys to move between columns, and `C` to create.
